Chernoff Bounds

Let \(X_i \sim N(\mu, \sigma^2)\) be independent random variables. Let \(X := \sum_i X_i\). $$\Pr(X/n - \mu \geq t) \leq e^{-\frac{nt^2}{2\sigma^2}}$$

Proof.

$$\Pr(X/n - \mu \geq t) = \Pr(e^{\lambda(X - n\mu)} \geq e^{n\lambdat})$$

Consider \(X_i, i = 1, \cdots, N\), follow a distribution with PDF \(f(x \mid \theta)\). Hence, the likelihood function is \(L(\theta) = \prod_{i=1}^N f(x_i \mid \theta)\). The score function \(s(\theta) = \frac{\partial}{\partial\theta}\log L(\theta)\).

Let \(\theta_0\) denote the true value of \(\theta\). We have $$s(\theta_0) \sim N(0,I(\theta_0)),$$ where \(I(\cdot)\) represents the Fisher Information.

This is a displayed formula: $$\int_{a}^{b} x^2 dx$$
